# 미션 - 페어매칭관리 애플리케이션
: 미션을 함께할 페어를 관리해주는 애플리케이션 

## 🚀 기능목록 
1. 기능 입력 받기.  
    1. 페어 매칭
        - .md 파일 받아오기 
        - 매칭 결과 저장
    2. 페어 조회   
        - 걸과에서 찾아서 조회
    3. 페어 초기화 
        - 결과에서 찾아서 초기화 
    Q. 종료   


2. 과정과 미션 출력하기.


3. 과정, 레벨, 미션 입력받기. 


4. 결과 출력하기. 
    -  1, 2 일시, 페어 매칭 결과 출력하기.
    -  3 일시, 초기화 되었습니다. 출력하기. 
    

5. 종료하기 전까지 위를 반복. 

### 예외 사항 
 1. 입력 값이 빈 값일 경우
 2. 기능 입력 시 1,2,3,Q 이외의 입력이 들어오는 경우.    
 2. 조회, 삭제 시 결과에 정보가 없는 경우 
 3. 매칭이 3회이상 이루어지지 않는 경우
